## 🔧 故事要从 GitHub Actions 说起

第一遍检查，`gh auth status`，认证正常。

第二遍查看，`gh run list`，找到问题了：workflow 里缺少 `permissions` 配置。

第三遍修改，添加：
```yaml
permissions:
  contents: write
  pages: write
```

然后推送：
```bash
git push origin main
```

✅ 修复完成！

> 感悟：GitHub 的坑，每个新手都要踩一遍。
